Introduction

to the objects of its environment to provide programmatic control over them.

Javascript And Java

avaScript, often abbreviated as JS, is a high-level, interpreted programming language primarily known for making web pages interactive. It was initially
designed for client-side web development, running directly within a user's web browser, but its capabilities have expanded.

Java is a widely used, high-level, object-oriented programming language and computing platform. Created in 1995 by Sun Microsystems (now owned by
Oracle), it is known for its "Write Once, Run Anywhere" (WORA) capability, which means compiled Java code can run on any device with a Java Virtual
Machine (JVM).

Variable

You use variables as symbolic names for values in your application. The names of variables, called identifiers, conform to certain rules.

A Javascript identifiers must start with a letter, underscore or dollar sign : subsequent characters can also be digits. Because Javascript is case sensitive ,
letters include the characters A through Z and characters a through z.

You can use ISO 8859-1 or Unicode letters such as a and u in identifiers. You can also use the Unicode escape sequences as characters in identifiers ,
Some examples of legal names are Number_hits, temp99, and _name.
